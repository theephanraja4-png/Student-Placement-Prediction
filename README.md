# 🎓 Student Placement Prediction using K-Nearest Neighbors (KNN)

A Machine Learning web application that predicts whether a student is likely to be placed based on academic performance and technical skills.

---

## 🚀 Features

- Student Placement Prediction
- K-Nearest Neighbors (KNN) Algorithm
- Interactive Streamlit Web Application
- Placement Probability Prediction
- Expected Salary Estimation
- Company Recommendations
- Jupyter Notebook for Model Training
- Synthetic Dataset Generation

---

## 🛠 Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly
- Joblib
- Jupyter Notebook

## 📊 Machine Learning Workflow

1. Generate Dataset
2. Data Preprocessing
3. Label Encoding
4. Feature Scaling
5. Train KNN Model
6. Save Model
7. Predict Placement using Streamlit

---

## ▶️ Run the Project

Install dependencies

```bash
pip install -r requirements.txt
```

Train the model

```bash
python train_model.py
```

Run the Streamlit application

```bash
streamlit run app.py
```
